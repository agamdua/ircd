# IRCd

A project of [The Great Code Club](http://www.greatcodeclub.com/).

A simple IRC server.

## Usage

    $ ./bin/ircd

## Installation

You need:

- A recent version of [node](http://nodejs.org/).

To install Node modules and compile the parser:

    $ npm install
